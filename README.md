# springboot-practices
Spring Boot learning repository: REST API, JPA, validation, security, and best practices.
## Tech stack
- Java 21
- Spring Boot
- MySQL
- JPA / Hibernate
- Postman

## Features
- User authentication (JWT)
- Product management
- Dynamic filtering

## Architecture
- Layered architecture

## Run project
- Run SQL scripts in resources/database/schema.sql
- Insert data via SQL script in resources/database/data
- Use Postman to test API
